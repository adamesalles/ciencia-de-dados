# Curso de Ciência de Dados

Bem-vindos ao curso de **Ciência de Dados**! Este repositório contém todo o material da disciplina, incluindo slides, notebooks interativos e exercícios práticos.

O curso é estruturado de forma **hands-on**, com forte foco em **manipulação de dados, engenharia de dados e construção de pipelines completos de análise**.

Ao longo das 15 semanas, os estudantes aprenderão a trabalhar com dados reais desde a coleta até a visualização e implantação de modelos.

# 🎯 Objetivos do Curso

Este curso tem como objetivo fornecer uma formação prática em **ciência de dados aplicada**.

Ao final do curso, os estudantes serão capazes de:

- Manipular e limpar bases de dados reais
- Trabalhar com diferentes formatos de dados (CSV, JSON, Parquet)
- Utilizar bibliotecas essenciais do ecossistema Python
- Construir pipelines de análise de dados
- Extrair dados da web e APIs
- Aplicar técnicas de clusterização e NLP
- Trabalhar com dados temporais
- Construir dashboards interativos
- Desenvolver projetos completos de ciência de dados

# 🧩 Avaliações

O curso possui duas avaliações principais:

### AV1 — Projeto de análise exploratória

Os estudantes receberão um conjunto de dados semi-estruturado e deverão:

- limpar e organizar os dados
- realizar análise exploratória
- aplicar técnicas básicas de modelagem ou clusterização

---

### AV2 — Projeto final

Os estudantes deverão construir um **pipeline completo de dados**, incluindo:

- coleta ou ingestão de dados
- processamento e limpeza
- feature engineering
- modelagem ou análise
- dashboard interativo
